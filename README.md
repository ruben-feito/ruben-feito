## Áreas de Especialización

**Desarrollo backend** — Laravel, CodeIgniter, Node.js. Diseño de APIs RESTful, autenticación segura (Sanctum, Passport, HMAC), procesamiento asíncrono con colas y optimización de rendimiento en entornos de producción.

**Integraciones eCommerce** — Desarrollo de conectores a medida para la sincronización de catálogos, stock y pedidos con marketplaces europeos y nacionales. Gestión de webhooks, control de errores y monitoreo en tiempo real.

**Frontend y paneles administrativos** — Astro.js, Next.js, Laravel Blade, Livewire y Tailwind CSS. Construcción de ERPs, CRMs y paneles de administración.

**Bases de datos** — MySQL, MariaDB, MongoDB. Diseño de esquemas eficientes, optimización de consultas complejas y gestión del rendimiento para aplicaciones escalables.

**Arquitectura de sistemas** — Microservicios, procesamiento en background, caching avanzado y webhooks. Integración con plataformas como Shopify, PrestaShop, WooCommerce y Odoo.
